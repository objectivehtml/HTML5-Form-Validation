Version 1.6.2 - 12/06/2012
-------------------------
Fixed a bug preventing the placeholder from triggering properly.

Version 1.6.1 - 2/23/2012
-------------------------
Fixed bugs that prevents script from working properly with appentToParents set to 'true'. Also fixed bug that prevented textareas from working properly.

Version 1.6 - 1/29/2012
-----------------------
Updated the way select elements are validated, giving them their own bind event sequence making the experience much more pleasant. Also updated the placeholder plugin to better represent the placeholder according to the W3C standard (like Webkit). Also changed the name of the plugin.


Version 1.4.6 - 5/2/2011
------------------------
A bug fix within the $._val() function preventing forms that had the reset value enabled from validating correctly with a placeholder value present.


Version 1.4.6 - 5/2/2011
------------------------

Version 1.4.5 - 4/21/2011

Version 1.4 - 4/1/2011
----------------------
Implemented a number of major bug fixes as the first the plugin was made available to the public. 1.4 marks the start of the first truly stable version. This is also the starting point of the change log.
